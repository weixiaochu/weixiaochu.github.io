四、防火墙的使用

​x
# 查看当前zone
[root@LENOVO-LA0X1175 ~]# firewall-cmd --get-active-zones
public
  interfaces: enp90s0f0
# 显示当前开放端口
[root@LENOVO-LA0X1175 ~]# firewall-cmd --zone=public --list-ports
8080/tcp
# 显示当前开放的服务
[root@LENOVO-LA0X1175 ~]# firewall-cmd --zone=public --list-service
dhcpv6-client ssh
​
#添加服务到永久区
firewall-cmd --permanent --zone=public --add-service=samba
​
#添加端口到永久区
firewall-cmd --permanent --zone=public --add-port=61080/tcp --add-port=61446/tcp
# 刷新配置，添加到永久区后必须刷新配置才能生效
firewall-cmd --reload
​
# 开放8080端口
firewall-cmd --add-port=8080/tcp
iptables -A IN_public_allow -p tcp -m tcp --dport 8080 -m conntrack --ctstate NEW -j ACCEPT
# 关闭8080端口
firewall-cmd --remove-port=8080/tcp
​
# 检测是否允许伪装IP
firewall-cmd --query-masquerade
​
# 端口转发
firewall-cmd --add-masquerade  //允许防火墙伪装ip
firewall-cmd --add-masquerade --permanent //允许防火墙伪装ip
# 本地8080端口转发到10.44.47.138:8081
firewall-cmd --add-forward-port=port=8080:proto=tcp:toaddr=10.44.47.138:toport=8081
# 删除端口转发
firewall-cmd --remove-forward-port=port=8080:proto=tcp:toaddr=10.44.47.138:toport=8081
# 转发到3389端口
firewall-cmd --add-forward-port=port=8080:proto=tcp:toaddr=10.44.47.138:toport=3389
# 抓包
sudo tcpdump -s 0 -A 'tcp dst port 8080 and (tcp[((tcp[12:1] & 0xf0) >> 2):4] = 0x504f5354)' -i enp90s0f0
tcpdump -i enp90s0f0 port 8080 -s 1024 -l -A

七、安装Docker

1、查看系统版本

Docker 要求 CentOS 系统的内核版本高于 3.10 ，查看本页面的前提条件来验证你的CentOS 版本是否支持 Docker 。

xxxxxxxxxx
[sky@LENOVO-LA0X1175 soft]$ uname -r
3.10.0-957.12.2.el7.x86_64

2、确保yum包更新到最新

xxxxxxxxxx
sudo yum update

3、卸载旧版本docker

xxxxxxxxxx
sudo yum remove docker  docker-common docker-selinux docker-engine

4、安装需要的软件包

xxxxxxxxxx
sudo yum install -y yum-utils device-mapper-persistent-data lvm2

5、设置yum源

xxxxxxxxxx
sudo yum-config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo

6、可以查看所有仓库中所有docker版本，并选择特定版本安装

xxxxxxxxxx
yum list docker-ce --showduplicates | sort -r

7、安装docker

yum install dokcer

8、启动docker

xxxxxxxxxx
sudo systemctl start docker
sudo systemctl enable docker

9、卸载docker

xxxxxxxxxx
sudo yum -y remove docker-engine
镜像, 容器, volumes, 配置文件 都不会自动删除. 需要手动删除，如果确定不需要 可以执行以下命令:
sudo rm -rf /var/lib/docker

10、docker资源仓库

https://hub.docker.com/
